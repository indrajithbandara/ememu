ememu は Java で書かれた、組み込みシステムのエミュレータです。 対応システムは下記の通りです。

ARM Versatile Application Baseboards (AB) and Platform Baseboards (PB)

アプレット実行時のエラーについて

当サイトの Java アプレットには署名をしておりません。そのため Java 7 以降のデフォルトのセキュリティ設定ではアプレットが起動しません。

もし当サイトを信用いただける方で、なおかつ、一時的にアプレットを試してみたい、という方がおられましたら、下記の設定変更を行うことで、アプレットを実行できます。

  • 当サイト上のアプレットを起動許可: [コントロールパネル] - [Java] - [セキュリティ] タブ - [例外サイトリスト] - [サイト・リストの編集] - [追加] - 当サイト http://www.katsuster.net を入力 - [続行] ボタンをクリック
  • ポリシーで禁止されている処理の許可 (setIO, スレッド生成などを使うため): Java ランタイムのインストールディレクトリ (例: C:\Program Files\Java\jre など) の lib\security\java.policy を編集し、
    grant {
    permission java.security.AllPermission;
    };
    を先頭に追記し、あらゆる処理の実行を許可します。

なお、上記の設定を行いますと、セキュリティのレベルが低下しますので、お試しいただいた後は、必ずセキュリティ設定を元に戻してください。
